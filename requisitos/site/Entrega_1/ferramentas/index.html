    Ferramentas

    Introdução

    Este documento tem o intuito de documentar todas as ferramentas utilizadas para a realização do projeto de requisitos de software, as ferramentas listadas aqui tem a intenção de dar suporte a documentação de requisitos do aplicativo **.

    Ferramentas utilizadas

    Logo Ferramenta Finalidade
    Github Manter o repositorio e hospedar
    Telegram Aplicativo escolhido para manter a comunicação do grupo.
    Youtube Utilizado para hospedar as gravações do projeto.
    VScode IDE utilizada para criar a git page do projeto.
    Teams Utilizado para realizar as reuniões e gravações das mesmas, além de realizar entrevistas.
    This person does not exist Utilizado para coletar imagens das personas.
    Lucidchart Criação do rich picture
    Figma Prototipação e o guia de estilo do projeto.
